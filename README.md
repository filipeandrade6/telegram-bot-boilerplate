# Boilerplate de bot para telegram

Boilerplate de bot para Telegram, utilizando Python, com comando de envio de arquivo, busca de termo em arquivo csv e lista de comandos.


## Requisitos

**Python-telegram-bot** - wrapper em python para controle de bot no Telegram.

`pip install python-telegram-bot`

**Poetry** - gerenciamento de ambiente de desenvolvimento.

`pip install poetry`
